About App

'Encrypto#tools' provide Tools to Decrypt or Encrypt your data using 128 bit encryption and with your own personal key.
Instruction:
Select Tool, Enter key and data hit Encrypt/Decrypt button, Copy the data using Copy button and you're set.
Note: key and data is not storing anywhere so 'KEEP KEY IN YOUR MIND'
*You can also use our android app 'Encrypto#Secure' which is a note app so that you can save your decrypted data as a note and only you can view, edit, delete the note using the key.
